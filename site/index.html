So we're expanding. Originally UXTOOLTIME was just for Axure widgets. Now we're branching out to Framer. We still love Axure but as we expand our skill sets we still like to share. So now we're including a few snippets for Framer.

Why Snippets you say? Because when we start a prototype we've found we like working in code to be the most flexible solution. So we created a series of snippets to add in basic system level things. You know that annoying status bar at the top with battery, wifi and clock. Pain in the backside to recreate each time so now it's a snippet. Just add and the code drops in.

We got tired of not having good axure widgets to design apps with. There are tons of quality PSDs and Sketch resources out there but just a lack of free axure resources. So we made our own, and guess what? We like to share.
